## Question 1

**In 2005, DigiCert became a founding member of which industry group?**

- A. CA/Browser Forum
- B. FIDO Alliance
- C. National Cybersecurity Alliance
- D. Open Web Application Security Project (OWASP)

**Correct answer:** A. CA/Browser Forum

## Question 2

**What industry first did DigiCert develop with Microsoft in 2007?**

- A. The first wildcard certificate
- B. The first certificate transparency log
- C. The first multi-domain certificate
- D. The first hardware-backed code signing certificate

**Correct answer:** C. The first multi-domain certificate

## Question 3

**What innovation did DigiCert build in 2013 that was the first of its kind to be accepted by Google?**

- A. Certificate Transparency log
- B. Wildcard certificate
- C. Post-quantum signing service
- D. Managed DNS platform

**Correct answer:** A. Certificate Transparency log

## Question 4

**In 2016, DigiCert introduced digital identity based on proven PKI standards for which emerging technology?**

- A. Drones
- B. Smart watches
- C. Autonomous submarines
- D. AR headsets

**Correct answer:** A. Drones

## Question 5

**In 2022, DigiCert's Root CA became the first approved for device attestation under which smart home standard?**

- A. Matter
- B. Zigbee
- C. Thread
- D. Z-Wave

**Correct answer:** A. Matter
